simple-server
=============

A simple web server written in C.

To install, run:

<pre>
git clone https://github.com/jns2/simple-server.git
</pre>
<pre>
make
</pre>

Then, to run the server:

<pre>
./simpleserver
</pre>

To test the server, either run:

<pre>
curl localhost:3500
</pre>

in a new terminal window, or point your browser to:

<pre>
localhost:3500
</pre>
